## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --save faculty
```

## Why is this needed?

The original math js library does not have a faculty function. Hence this repo.

## Usage

```js
const faculty = require("faculty");

faculty(6);
```
